FinSight Screener is a Streamlit-based stock screening application designed to help users identify U.S. stocks that meet specific dividend, fundamental, and technical criteria.  
The app integrates financial statement data with technical indicators to support data-driven investment screening.

---

## Key Features

- Upload or generate stock datasets for screening
- Filter stocks based on:
  - Dividend consistency and payout conditions
  - Fundamental metrics such as earnings, equity, and debt trends
  - Technical indicators including MACD
- Interactive filtering interface built with Streamlit
- Automatically generate MACD charts for selected tickers
- Export filtered results as CSV for further analysis

---

## App Workflow

1. Load or upload stock data
2. Apply user-defined screening conditions
3. Review the filtered stock list
4. Generate technical indicator charts (MACD)
5. Download filtered results for external analysis
